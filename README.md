# Análise de Dados

## Objetivo
Analisar dados de aluguel de bicicletas em Seul para identificar padrões de uso e fatores que influenciam a demanda.

## Tecnologias
- Python, Pandas
- Matplotlib, Seaborn
- Scikit-learn, XGBoost
- Pipeline de dados
- Jupyter Notebook

## Análises realizadas
- Tratamento de dados
- Análise exploratória (EDA)
- Modelagem preditiva
- Avaliação de modelo

## Insights
- Influência dos dias da semana e do clima na demanda
- Relação entre temperatura e aluguel

## Estrutura
- data/: base de dados
- notebooks/: análises em Python
- analises-html/: conteúdo das análises, para visualização em html.
